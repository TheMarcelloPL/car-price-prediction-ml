# Car Price Prediction ML

Projekt indywidualny z przedmiotu Uczenie Maszynowe.

## Autor

Marcel Szulc

## Cel projektu

Celem projektu jest przewidywanie ceny samochodu na podstawie jego parametrów technicznych i informacji z ogłoszenia. Jest to problem regresji, ponieważ model przewiduje konkretną wartość liczbową, czyli cenę.

## Dane

Projekt korzysta ze zbioru Automobile z UCI Machine Learning Repository.

Dane zawierają informacje o samochodach, między innymi:
- marka,
- typ paliwa,
- rodzaj nadwozia,
- masa samochodu,
- pojemność silnika,
- moc,
- spalanie,
- cena.

## Modele

W projekcie porównano kilka modeli regresyjnych:
- Linear Regression,
- Ridge Regression,
- Decision Tree,
- Random Forest,
- Gradient Boosting,
- KNN Regression.
